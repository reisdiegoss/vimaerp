## Goal

Gerar interfaces limpas, modernas e responsivas utilizando React, Tailwind CSS e componentes corporativos da biblioteca Preline UI.

## Instructions

* Escreva código JSX/TSX semântico utilizando classes utilitárias do Tailwind CSS (usando `className`).
* Utilize as estruturas do Preline UI para lidar com alta densidade de dados corporativos (ex: Data Tables ricas, Modais, Abas, Selects Múltiplos).
* Use a biblioteca Phosphor Icons para toda a iconografia (ex: `<i className="ph ph-icon"></i>`).
* Aplique princípios de design corporativo: bordas arredondadas suaves (`rounded-lg` ou `rounded-xl`), sombras sutis (`shadow-sm`) e amplo uso de whitespace (paddings e margins consistentes).

## Constraints

* DO NOT use Bootstrap, AdminLTE, Bulma, MUI ou qualquer outro framework CSS. Utilize ESTRITAMENTE Tailwind CSS.
* DO NOT escreva textos na interface em outro idioma que não seja Português do Brasil.
* DO NOT use o atributo HTML `class=`; como estamos no ambiente React, utilize obrigatoriamente `className=`.
